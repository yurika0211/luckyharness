# v0.22.0 PRD — 多 Agent 协作

**目标**：实现 Agent-to-Agent 通信、任务委派与结果聚合，让多个 Agent 协同完成复杂任务（如并行 RAG、Debate 式决策、Pipeline 工作流）。

**核心特性**
- **通信协议**：JSON + protobuf 兼容的消息格式（Task, Result, Status, Error）
- **Agent Registry**：增强发现、健康检查、能力匹配
- **Task Delegation**：拆分、委派、超时、重试、取消
- **Result Aggregation**：复用已有 Aggregator，支持 Pipeline/Parallel/Debate
- **协作模式**：串行、并行、辩论
- **API + REPL**：/api/v1/agents/delegate + lh agent delegate/list/cancel
- **测试**：30+ 单元/集成测试

**文件规划**
- internal/agent/protocol.go（新）
- internal/agent/delegator.go（新）
- internal/collab/registry.go（扩展）
- internal/collab/aggregator.go（已复用）
- API 路由 + REPL 命令
